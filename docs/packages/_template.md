# packages 共享包文档模板

> 占位。`packages/` 实际包(自定义 eslint 规则、前后端共享依赖)出现时,复制本模板到 `docs/packages/<package>.md` 填写。

## 计划范围

- 自定义 eslint 规则包(`packages/eslint-config` 或类似)
- 前后端共享类型/工具包(`packages/shared`)

## 每个包文档应含

- 包名与职责
- 依赖方(backend / frontend / both)
- API/导出
- 开发与发布流程
- 与 eslint config / tsconfig 的关系
